#479 – Dave Plummer

Programming, Autism, and Old-School Microsoft Stories

核心思想（3 点）

  • 编程生涯与自闭谱系等神经多样性可以相互成就，而非互斥。
  • 早期微软时代的工程文化深刻影响了 PC 生态与开发者习惯。
  • 个人故事与底层细节，是理解技术史与职场文化的最佳切入点。

重要纪实 / 历史启示

  • Dave 参与开发 Windows 95/NT/XP，被广泛认为是 Task Manager 的主要作者之一。
  • 资源极度受限的时代所做的架构与 API 决策，对后世软件生态产生了长期锁定效应。
  • 自闭谱系特征（高专注、模式敏感、对细节的痴迷）在系统级工程中可能成为优势。
  • 职场对神经多样性的包容与支持，直接决定人才能否被“看见”和被善用。
  • 老微软文化的高压与 ownership 精神，为今天的软件团队提供正反两面的借鉴。
  • 好的工具（如 Task Manager）改变了普通用户理解和管理系统的方式。
本页为非官方个人整理，基于公开访谈主题做内容提炼，供学习与讨论使用。
