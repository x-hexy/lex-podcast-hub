# Lex Podcast Hub

非官方的 Lex Fridman Podcast 精选访谈内容整理。本项目旨在提供一个干净、现代的界面来阅读和探索播客的核心思想。

## 在线访问

[点击这里访问在线版本](https://x-hexy.github.io/lex-podcast-hub/) (请替换为实际的 GitHub Pages 链接)

## 功能特点

- **精选内容**：每期播客提炼 3 点核心思想 + 5-10 条重要纪实。
- **现代设计**：暗色模式优先，Glassmorphism 风格，响应式布局。
- **交互体验**：
  - 折叠/展开所有卡片
  - 快捷回到顶部
  - 平滑滚动导航
- **模块化结构**：HTML, CSS, JS 分离，易于维护和扩展。

## 本地开发

1. 克隆仓库：
   ```bash
   git clone https://github.com/x-hexy/lex-podcast-hub.git
   ```
2. 直接在浏览器中打开 `index.html` 即可预览。

## 部署

本项目是一个静态网站，可以轻松部署到 GitHub Pages。

1.而在 GitHub 仓库的 Settings -> Pages 中。
2. 选择 `main` 分支作为源。
3. 保存即可。

## 目录结构

- `index.html`: 首页
- `assets/`: 静态资源 (CSS, JS)
- `episodes/`: 单集页面 (示例)
